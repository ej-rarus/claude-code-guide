빠른 시작

4단계로 Claude Code 문서 작업을 시작하세요.

1 작업 폴더 지정

먼저 작업할 폴더 경로를 알려주세요.

명령어
"/Users/username/Desktop/project 경로에서 작업할거야"

2 파일 확인

해당 폴더에 어떤 파일이 있는지 확인합니다.

명령어
"이 폴더에 뭐 있어?"

3 작업 요청

원하는 작업을 자연어로 요청하세요.

명령어
"이 PPT 파일 마크다운으로 변환해줘"

4 결과 활용

결과물을 클립보드에 복사하거나 파일로 저장합니다.

명령어
"클립보드에 복사해줘"

Pro Tip

클립보드에 복사한 내용은 클릭업, 노션, 슬랙 등에 바로 붙여넣기(Cmd+V) 가능합니다.

PPT → Markdown 변환

파워포인트 파일의 텍스트를 추출하여 마크다운 문서로 변환합니다.

기본 명령어

예시
# 기본 변환
"이 PPT 파일 마크다운으로 변환해줘"

# 특정 파일 지정
"바이오스펙트럼_디자인요청서.pptx 내용 MD로 정리해줘"

# 특정 슬라이드만
"PPT에서 3~5번 슬라이드만 추출해줘"

변환 결과물

PPT 요소 변환 결과
슬라이드 제목 # 제목 또는 ## 소제목
본문 텍스트 일반 텍스트 또는 리스트
표 마크다운 테이블
이미지 이미지 설명 텍스트

참고

PPT 내의 이미지는 텍스트로 변환되지 않지만, 이미지 파일을 별도로 분석해달라고 요청하면 내용을 설명해드릴 수 있습니다.

PDF → Markdown 변환

PDF 문서의 텍스트와 구조를 추출하여 마크다운으로 변환합니다.

기본 명령어

예시
# 전체 변환
"이 PDF 읽고 마크다운으로 정리해줘"

# 핵심만 추출
"계약서.pdf 핵심 내용만 추출해서 문서화해줘"

# 특정 페이지
"PDF 5~10페이지만 정리해줘"

주의사항

스캔된 이미지 형태의 PDF는 OCR 처리가 필요할 수 있으며, 복잡한 레이아웃은 수동 조정이 필요할 수 있습니다.

이미지 → 텍스트 (OCR)

스크린샷, 사진 등 이미지 속 텍스트를 추출하고 분석합니다.

기본 명령어

예시
# 텍스트 추출
"이 스크린샷에 있는 텍스트 추출해줘"

# 표 변환
"이미지 속 표 내용을 마크다운 테이블로 만들어줘"

# 이미지 분석
"이 이미지가 무엇을 보여주는지 설명해줘"

지원 형식

PNG

JPG

GIF

WebP

Excel/CSV → 테이블 변환

스프레드시트 데이터를 마크다운 테이블로 변환합니다.

기본 명령어

예시
# Excel 변환
"이 엑셀 파일 마크다운 표로 변환해줘"

# CSV 변환
"data.csv 내용을 보기 좋게 정리해줘"

# 특정 시트
"Sheet2의 데이터만 테이블로 만들어줘"

새 문서 생성

다양한 형식의 문서를 자동으로 생성합니다.

문서 유형별 명령어

예시
# 회의록
"회의록 템플릿 만들어줘"

# 기획서
"프로젝트 기획서 초안 작성해줘"

# API 문서
"API 문서 형식으로 정리해줘"

# README
"이 프로젝트용 README.md 만들어줘"

생성 가능한 문서 유형

회의록

참석자, 안건, 논의사항, 액션아이템 등을 체계적으로 정리

기획서

목표, 범위, 일정, 리소스 등을 포함한 프로젝트 기획 문서

기술 문서

API 문서, 아키텍처 문서, 설치 가이드 등

버그 리포트

재현 단계, 예상 동작, 실제 동작, 환경 정보 등

문서 수정

기존 문서의 내용을 수정하고 개선합니다.

수정 명령어

예시
# 섹션 추가
"README.md에 설치 방법 섹션 추가해줘"

# 구조 변경
"이 문서 목차 구조 다시 잡아줘"

# 내용 수정
"오타 수정하고 문장 다듬어줘"

# 형식 변환
"이 리스트를 표 형식으로 바꿔줘"

템플릿 활용

자주 사용하는 문서 템플릿을 생성하고 활용합니다.

템플릿 생성

예시
# 주간 보고서
"주간 보고서 템플릿 만들어줘"

# 이슈 템플릿
"GitHub 이슈 템플릿 만들어줘"

# PR 템플릿
"Pull Request 템플릿 생성해줘"

문서 분석

문서 요약, 핵심 추출, 비교 분석 등을 수행합니다.

분석 명령어

예시
# 요약
"이 문서 3줄로 요약해줘"
"긴 보고서 핵심만 정리해줘"

# 추출
"회의록에서 액션 아이템만 추출해줘"
"이 문서에서 날짜/일정 관련 내용만 뽑아줘"

# 비교
"두 문서 차이점 비교해줘"
"버전1과 버전2 변경사항 정리해줘"

분석 유형

유형 설명 예시 명령
요약 긴 문서의 핵심 내용을 짧게 정리 "3문장으로 요약해줘"
추출 특정 정보만 선별하여 추출 "담당자 이름만 추출해줘"
비교 두 문서 간 차이점 분석 "v1과 v2 차이점 알려줘"
검증 문서 내 오류나 모순 확인 "이 문서에 모순되는 내용 있어?"

웹 정보 수집

웹 페이지 분석과 최신 정보 검색을 통해 문서를 작성합니다.

웹 관련 명령어

예시
# URL 분석
"이 URL 내용 요약해줘: https://example.com"
"경쟁사 웹사이트 3개 비교 분석해줘"

# 정보 검색
"2024년 최신 React 트렌드 조사해서 정리해줘"
"Shopify 테마 추천 검색해서 문서로 만들어줘"

제한사항

로그인이 필요한 페이지나 비공개 콘텐츠는 접근할 수 없습니다. 공개된 웹 페이지만 분석 가능합니다.

자동화

반복 작업을 자동화하여 효율을 높입니다.

자동화 명령어

예시
# 파일 병합
"이 폴더의 모든 md 파일 하나로 합쳐줘"

# 일괄 분석
"images 폴더 이미지들 분석해서 목록 만들어줘"

# 일괄 수정
"모든 문서에서 '주식회사'를 '(주)'로 변경해줘"

# 파일명 변경
"파일명에 날짜 prefix 붙여줘"

자동화 가능한 작업

여러 파일 일괄 변환
텍스트 일괄 치환
파일명 일괄 변경
폴더 구조 생성
데이터 추출 및 정리
문서 병합

실전 예시

실제 업무에서 활용할 수 있는 예시들입니다.

회의록 정리

입력 예시
"오늘 회의 내용이야:
- 참석자: 김대리, 박과장, 이부장
- 주제: 신규 프로젝트 킥오프
- 논의사항: 일정, 예산, 담당자 배정
- 결정사항: 12월 말까지 1차 완료
회의록 형식으로 정리해줘"

기획서 작성

입력 예시
"다음 내용으로 기획서 작성해줘:
- 프로젝트명: 쇼핑몰 리뉴얼
- 목표: UX 개선 및 전환율 향상
- 범위: 메인페이지, 상품상세, 장바구니
- 대상: 20-40대 여성
마크다운으로 섹션 나눠서 작성해줘"

API 문서화

입력 예시
"이 JSON 데이터를 API 문서 형식으로 정리해줘"

"DB 스키마 보고 ERD 설명 문서 만들어줘"

"이 함수들 JSDoc 형식으로 문서화해줘"

경쟁사 분석

입력 예시
"다음 3개 웹사이트 비교 분석해서 표로 정리해줘:
- https://competitor1.com
- https://competitor2.com
- https://competitor3.com
UI/UX, 기능, 가격 정책 위주로"

유용한 팁

Claude Code를 더 효율적으로 사용하는 방법들입니다.

팁 명령어 예시 설명
작업 경로 지정 "~/Desktop/project 경로에서 작업할거야" 모든 작업의 기준 폴더 설정
표 형식 출력 "표 형식으로 정리해줘" 데이터를 보기 좋은 테이블로
클립보드 복사 "클립보드에 복사해줘" 결과물 바로 붙여넣기 가능
파일명 지정 "파일명은 YYYYMMDD_제목.md로 해줘" 원하는 형식으로 저장
출력 위치 지정 "output 폴더에 저장해줘" 결과물 저장 위치 지정

터미널 클립보드 복사 (Mac)

cat 파일명.md | pbcopy 명령으로 파일 내용을 클립보드에 바로 복사할 수 있습니다.
